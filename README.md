# Projeto Acervo de Livros

Este projeto foi feito para gerenciar um acervo de livros via console.

### Funcionalidades

- Cadastrar livros (título, autor, ano e editora)
- Listar todos os livros cadastrados
- Buscar livros por:
  - Autor
  - Ano de publicação
  - Palavra no título
